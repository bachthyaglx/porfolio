## Getting Started
First, install node modules:
```bash
npm install
```

Second, run the server, you have 2 options:
* Run development server:
```bash
npm run dev
```
* Run production server:
```bash
npm run prod
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
